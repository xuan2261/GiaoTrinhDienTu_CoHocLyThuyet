# Simulation Rewrite V2

## Overview
This plan outlines the complete teardown and rebuild of the theoretical mechanics simulation system. It replaces the complex 5-layer custom physics/canvas engine with a robust architecture using headless Matter.js for physics, SVG/DOM for crisp rendering, Chart.js for realtime data visualization, and standardized native HTML5 controls for unified UI.
