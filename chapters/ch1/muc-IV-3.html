Kết quả thu gọn hệ lực về một tâm có thể xảy ra các trường hợp sau:

Hệ lực đồng quy tại O

Vì M O → = ∑ k = 1 n M O → ( F → k ) = 0 nên M O →  . R ' → = 0 . Vậy hệ lực đồng quy hoặc cân bằng hoặc có hợp lực.

Hệ ngẫu lực

Véc tơ chính của hệ ngẫu lực luôn luôn bằng không.

R ' → = ∑ k = 1 n F → k = 0

(.)

Nên hệ ngẫu lực hoặc cân bằng hoặc song song hoặc tương đương với một ngẫu lực.

Hệ lực song song

Véc tơ chính của hệ lực song song R ' → = ∑ k = 1 n F → k có phương song song với các lực thành phần. Ngoài ra, M → O ( F → k ) vuông góc với lực F → k , tức vuông góc với phương của véc tơ chính R → ' . Do đó mô men chính của hệ lực: M → O = ∑ k = 1 n M → O ( F → k ) cũng vuông góc với phương của véc tơ chính R → ' . Vậy: M → O . R → ' = 0 → . Do đó các hệ lực song song có các dạng chuẩn là: cân bằng ngẫu lực, hợp lực. Trong trường hợp hệ lực song song cùng chiều, véc tơ chính: R ' → = ∑ k = 1 n F → k ≠ 0 nên hệ chỉ có một dạng chuẩn, là hợp lực.

Hệ lực phẳng

Hệ lực phẳng cân bằng khi véc tơ chính và mô men chính của hệ lực đối với một điểm đều bằng không, tức:

R ' → = ∑ k = 1 n F → k = 0 ; M O → = ∑ k = 1 n M O → ( F → k ) = 0

(.)

Hệ lực phẳng tương đương với một ngẫu lực khi véc tơ chính của hệ lực bằng không, còn mô men chính của hệ lực đối với một điểm khác không.

R ' → = ∑ k = 1 n F → k = 0 ; M O → = ∑ k = 1 n M O → ( F → k ) ≠ 0

(.)

Hình 1.20. Hệ lực phân bố

Hệ lực phân bố có thể xem là trường hợp giới hạn của hệ lực tập trung (hệ lực phẳng song song cùng chiều), với các lực tập trung F → k :

Fk = DQk = q(x’k).Dxk với xk < x’k < xk+1

Véc tơ chính của hệ lực song song cùng chiều, cùng phương với các lực F → k và có giá trị:

R ' = ∑ F k = l i m ∑ q ( x ' k ) Δ x k = ∫ 0 l q ( x ) d x

. Chú ý rằng tích phân trên biểu diễn diện tích của biểu thức phân bố lực. Mô men chính của hệ lực ( F → k ) đối với một điểm nào đó, chẳng hạn đối với điểm đầu mút của dầm sẽ bằng:

\[\vec{M}_O=\sum_{k=1}^{n}\vec{M}_O(\vec{F}_k)=\lim\sum q(x'_k)x_k\Delta x_k=\int_0^l q(x)x\,dx\]

Vậy hệ lực phân bố có hợp lực R → hướng thẳng đứng xuống có trị số:

R = ∫ 0 1 q ( x ) d x

(.)

Và cách đầu mút của dầm một đoạn d:

d = ∫ 0 l q ( x ) x d x ∫ 0 l q ( x ) d x

(.)

Chú ý: Xét hai trường hợp riêng sau:

  • Cường độ phân bố lực đều, tức q(x) = q0 = hằng như hình 1.21.

Hình 1.21. Hệ lực phân bố đều

Trị số của hợp lực tính theo công thức (1.14) bằng: R = ∫ 0 l q 0 d x = q 0 . l . Và hợp lực cách đầu mút dầm một đoạn d:

D = ∫ 0 l q 0 x d x q 0 . l = l 2

Vậy hợp lực đặt tại điểm giữa của dầm và có giá trị bằng diện tích của hình chữ nhật phân bố lực.

  • Cường độ phân bố lực tuyến tính

Giả sử lực phân bố dọc dầm theo luật tam giác có đáy là q0. Rõ ràng: q ( x ) = q 0 x l

Hình 1.22. Hệ lực phân bố tuyến tính

Công thức (1.14) cho ta trị số của hợp lực \(\vec{R}\): R = ∫ 0 l q 0 x l d x = 1 2 q 0 l

Và hợp lực cách đầu mút một đoạn d:

d = ∫ 0 l q 0 x l x d x 1 2 q 0 . l = q 0 l 2 3 1 2 q 0 . l = 2 3 l

(.)

Vậy hệ lực phân bố tam giác có hợp lực có trị số bằng R = 1 2 q 0 l (diện tích tam giác phân bố lực) và cách đỉnh của tam giác phân bố lực một đoạn bằng 2 3 l (tức đi qua tâm của tam giác phân bố lực) . Qua hai thí dụ về phân bố lực hằng và lực tuyến tính và công thức (1.15) và (1.16) có thể đi đến kết luận sau: Hệ lực phân bố có hợp lực có giá trị bằng diện tích (với tỉ lệ xích nào đó) của biểu đồ phân bố lực và đi qua trọng tâm của biểu đồ đó.
